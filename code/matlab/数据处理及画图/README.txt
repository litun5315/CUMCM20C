data3calculate.m
作用：计算不同自然增长率在不同年利率下对年平均收益的影响并画图
使用方法：在MATLAB2019a以及以上环境下可直接运行，确保同目录下有文件“附件三.xlsx”（原题目附带的第三个文件）
data3plot.m
作用：直接对附件三的数据进行画图分析
使用方法：在MATLAB2019a以及以上环境下可直接运行，确保同目录下有文件“附件三.xlsx”（原题目附带的第三个文件）
data2.m
作用：对附件二的数据进行一部分预处理
使用方法：在MATLAB2019a以及以上环境下可直接运行，确保同目录下有文件“附件二.xlsx”（原题目附带的第二个文件）
Q3Filereader.m
作用：处理在第三问中企业与上下流企业之间的交易次数和交易额度构成相关矩阵
使用方法：使用方法：在MATLAB2019a以及以上环境下可直接运行，确保同目录下有文件“data2.xlsx”（简化后保留有效交易的企业上下流交易对象和额度的附件二）和“行业分类.xlsx”(正则分词后语言匹配后对附件二进行的行业分类表)
注意：此处构建多个大小约为302*400000+的矩阵，对于配置较低设备上的MATLAB运行时需要调整软件内存设置
connection.m
作用：判断附件二中企业和任意上下流企业是否有合作
使用方法：此代码是一个函数，需要在运行Q3Filereader.m之后再运行，输入参数分别为“上流相关矩阵”“下流相关矩阵”“企业序号”“上下流企业序号”“上流还是下流（0代表寻找上流企业，1代表寻找下流）”，输出1代表有合作，0代表无合作
